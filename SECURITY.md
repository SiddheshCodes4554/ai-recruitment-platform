# Security Policy

## Supported Versions

We currently support and maintain security updates for the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a potential security vulnerability, please do not open a public issue. Instead, email a report to `participant@example.com`.

Please include in your report:
* A detailed description of the vulnerability and its potential impact.
* Steps to reproduce the issue (including sample JD text or candidate payloads).
* Any potential mitigations you've identified.

We will acknowledge receipt of your vulnerability report within 48 hours and work with you to coordinate a security patch release.
